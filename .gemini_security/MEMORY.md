# Active Games Page Rebuild Strategy

## Core Objectives (Based on Architecture Doc)
- Simple, exciting, and automated.
- High-fidelity UI/UX with motion, audio, and celebration effects.
- Dynamic backgrounds and business storefront themes.
- End-to-end experience from campaign selection to reward claim.

## UI/UX Elements to Implement
- **Visual Layers:** Dynamic backgrounds, storefront themes, animated boxes, rotating rewards.
- **Motion Effects:** Box shaking, reward swapping, reveal transitions, glow effects, countdown animations.
- **Audio Layer:** Win sounds, music library, volume controls.
- **Celebration Layer:** Fireworks, confetti, flash effects, animated win cards.

## Technical Stack
- Next.js (App Router)
- Tailwind CSS
- Framer Motion (Advanced animations)
- Lucide React (Icons)
- Canvas Confetti (Celebration)
- Zustand (Store management)
